# Projeto Social

Projeto desenvolvido com foco em impacto social, criado para aplicar na prática conceitos de desenvolvimento front-end e organização de interfaces voltadas ao usuário final.

## 🎯 Objetivo
Simular uma aplicação real utilizada por projetos sociais, com foco em clareza, acessibilidade e organização da informação.

## 🛠 Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript

## 📌 Funcionalidades
- Layout responsivo
- Estrutura semântica
- Organização clara do conteúdo
- Interface voltada à experiência do usuário

## 🚀 Aprendizados
- Estruturação de páginas reais
- Organização de layout
- Boas práticas de front-end
- Versionamento com Git e GitHub
